# CodeAlpha AI Chatbot

## Description
A simple Java-based AI Chatbot that interacts with users and provides predefined responses.

## Features
- Interactive conversation
- Rule-based responses
- Handles common questions
- Exit command using "bye"

## Technologies Used
- Java
- OOP Concepts
